JILCROW
=======

A static site generator, tailored for *my* own needs. (Work in progress.)

Use It
------

    mkdir mysite
    cd mysite
    mkvirtualenv mysite
    pip install -e http://bitbucket.org/sjl/jilcrow/#egg=jilcrow

    ... make your folders ...

    jilcrow .
